🧠 Reconfigurable FPGA CNN Accelerator

Built a reconfigurable FPGA accelerator for lightweight CNN inference targeting edge AI applications.

  • 16-lane parallel compute engine with INT8 quantization
  • 128-bit BRAM-based memory system with efficient data reuse
  • AXI CDMA-based data movement for high-speed DDR ↔ PL transfers
  • Supports convolution, depthwise convolution, pooling & activation

Tech: Verilog (RTL), AXI4/AXI-Lite, AXI CDMA, PYNQ (Python), Vivado

⚡ FPGA Acceleration of 2D Convolution (GeMM-Based)

  • FSM-driven dataflow (IPICU) for structured execution
  • Optimized memory access, line buffering, and patch reuse

  • Scalable architecture for larger inputs and multiple kernels

⚡ 3×3 GEMM Accelerator in Verilog

Built a hardware-accelerated 3×3 GEMM engine in Verilog for efficient matrix multiplication on FPGA.

  • BRAM-based architecture for row/column data streaming
  • FSM-controlled pipeline for coordinated compute and memory access
  • Parallel MAC operations for low-latency computation
  • Hardware validation using Integrated Logic Analyzer (ILA)

Tech: Verilog (RTL), FPGA Design, BRAM Architecture, FSM Design, Vivado, ILA
