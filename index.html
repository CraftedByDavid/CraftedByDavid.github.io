🚀 Featured Projects

🧠 Reconfigurable FPGA CNN Accelerator

Designed and implemented a reconfigurable FPGA-based accelerator for lightweight CNN inference targeting edge devices. The architecture features a 16-lane parallel compute engine, 128-bit BRAM-based memory system, and INT8 quantization for high efficiency.

Optimized data movement using AXI CDMA, enabling high-speed DDR-to-BRAM transfers and significantly reducing CPU overhead. The design supports convolution, depthwise convolution, pooling, and activation operations.

Achieved up to 3146× speedup over CPU inference and ~11× improvement compared to MMIO-based FPGA execution, demonstrating real-time performance for edge AI applications.

Tech: Verilog (RTL), AXI4/AXI-Lite, AXI CDMA, PYNQ (Python), Vivado
