🚀 Featured Projects

🧠 Reconfigurable FPGA CNN Accelerator

Designed and implemented a reconfigurable FPGA-based accelerator for lightweight CNN inference targeting edge devices. The architecture features a 16-lane parallel compute engine, 128-bit BRAM-based memory system, and INT8 quantization for high efficiency.

Optimized data movement using AXI CDMA, enabling high-speed DDR-to-BRAM transfers and significantly reducing CPU overhead. The design supports convolution, depthwise convolution, pooling, and activation operations.

Achieved up to 3146× speedup over CPU inference and ~11× improvement compared to MMIO-based FPGA execution, demonstrating real-time performance for edge AI applications.

Tech: Verilog (RTL), AXI4/AXI-Lite, AXI CDMA, PYNQ (Python), Vivado

⚡ FPGA Acceleration of 2D Convolution (GeMM-Based)

Built a high-performance FPGA accelerator for 2D convolution by transforming convolution into matrix multiplication (im2col + GeMM), enabling efficient parallel execution on hardware.

Designed a custom FSM-driven dataflow architecture (IPICU) with optimized memory access, line buffering, and patch reuse to minimize bandwidth overhead and maximize throughput.

Implemented and evaluated multiple execution modes: full-frame (resource-efficient), segmented-frame (high parallelism), and multi-kernel (throughput scaling).

Achieved up to 10–16× speedup using segmentation and ~3× acceleration over ARM-based processing systems, demonstrating the effectiveness of FPGA-based CNN acceleration.

Tech: Verilog (RTL), FPGA (Zynq UltraScale+, Spartan-7), Vivado, GeMM (im2col), Hardware Dataflow Design

⚡ 3×3 GEMM Accelerator in Verilog

Designed and implemented a hardware-accelerated 3×3 matrix multiplication (GEMM) engine using Verilog, optimized for efficient execution on FPGA using on-chip memory.

Built a BRAM-centric architecture where matrix rows and columns are streamed from dedicated memory blocks, enabling structured data access and minimizing memory bottlenecks.

Developed an FSM-controlled compute pipeline to orchestrate memory reads, parallel multiply-accumulate (MAC) operations, and output storage with precise timing control.

Verified functionality on hardware using Integrated Logic Analyzer (ILA), ensuring correct computation and real-time observability of internal signals.

Tech: Verilog (RTL), FPGA Design, BRAM Architecture, FSM Design, Vivado, ILA
