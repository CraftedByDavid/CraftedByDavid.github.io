🚀 Featured Projects

🧠 Reconfigurable FPGA CNN Accelerator

Designed and implemented a reconfigurable FPGA-based accelerator for lightweight CNN inference targeting edge devices. The architecture features a 16-lane parallel compute engine, 128-bit BRAM-based memory system, and INT8 quantization for high efficiency.

Optimized data movement using AXI CDMA, enabling high-speed DDR-to-BRAM transfers and significantly reducing CPU overhead. The design supports convolution, depthwise convolution, pooling, and activation operations.

Achieved up to 3146× speedup over CPU inference and ~11× improvement compared to MMIO-based FPGA execution, demonstrating real-time performance for edge AI applications.

Tech: Verilog (RTL), AXI4/AXI-Lite, AXI CDMA, PYNQ (Python), Vivado

⚡ Hardware Acceleration of 2D Convolution using GeMM

Designed and implemented an FPGA-based acceleration approach for 2D convolution using the GeMM (General Matrix Multiplication) transformation. The design converts convolution operations into matrix multiplication to exploit parallel computation on FPGA.

Developed an optimized dataflow architecture with efficient memory access patterns, enabling high-throughput computation. Focused on improving performance through parallelism, data reuse, and efficient buffering.

This work demonstrates a scalable approach for accelerating CNN workloads and serves as a foundation for high-performance deep learning accelerators.

Tech: Verilog (RTL), FPGA Design, Matrix Multiplication (GeMM), Vivado
